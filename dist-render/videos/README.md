# Videos de Fondo

Coloca aquí el video de workflows y procesos empresariales.

## Requisitos del video:
- Formato: MP4 (H.264) y/o WebM para mejor compatibilidad
- Nombre del archivo: `workflow-processes.mp4` y/o `workflow-processes.webm`
- Duración: Recomendado 30-60 segundos (se reproducirá en loop)
- Resolución: Mínimo 1920x1080 para buena calidad
- Contenido: Workflows, procesos empresariales, automatización, flujos de trabajo

## Ejemplos de contenido:
- Diagramas de flujo animados
- Procesos de automatización
- Workflows empresariales
- Flujos de datos
- Procesos de negocio visualizados

El video se mostrará con opacidad reducida y blur sutil para no interferir con el contenido principal.
